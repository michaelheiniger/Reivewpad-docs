**Description**:

Adds a message to the warnings section of the report.

**Parameters**:

| Parameter  | Type    | Description                               |
| :--------- | :------ | :---------------------------------------- |
| `comment`  | string  | The comment to be added in the warnings.  |

**Examples**:

```yml
$warn("Please do not forget to add the tests.")
```

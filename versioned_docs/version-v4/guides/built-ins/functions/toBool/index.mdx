**Description**:

Converts a string into a boolean. This built-in is a wrapper around [strconv.ParseBool](https://pkg.go.dev/strconv#ParseBool).
It accepts 1, t, T, TRUE, true, True, 0, f, F, FALSE, false, False. Any other value returns an error.

**Parameters**:

| Parameter | Type   | Description                                    |
| :-------- | :----- | :--------------------------------------------- |
| `string`  | string | The string value to be converted to a boolean. |

**Return value**:

| Type   | Description           |
| :----- | :-------------------- |
| `bool` | The converted boolean |

**Examples**:

```yml
$toBool("true")
$toBool("false")
$toBool($selectFromContext("$.locked"))
```
